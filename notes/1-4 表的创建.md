# 1-4 表的创建

[toc]

## 数据库的创建（CREATE DATABASE语句）

**语法1-1 创建数据库的CREATE DATABASE语句**

```sql
CREATE DATABASE <数据库名称>;
```

## 表的创建（CREATE TABLE语句）

**语法1-1 创建表的CREATE TABLE语句**

```sql
CREATE TABLE <表名>
(<列名1> <数据类型> <该列所需的约束>,
 <列名2> <数据类型> <该列所需的约束>,
 <列名3> <数据类型> <该列所需的约束>,
 <列名4> <数据类型> <该列所需的约束>,
 .
 .
 .
 <该表的约束1>, <该表的约束2>, ......);
```
